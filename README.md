# Website Purpose
I created a multi-page website that provides information about my favorite places and ways to ski. The purpose of this site is to share my passion for skiing and highlight the locations and styles of skiing that are most important to me. The website focuses on Vermont ski culture, including Mad River Glen, Smuggler's Notch, and backcountry skiing.

# Key Features
## Multi-page Structure
- Home page (About Me and my skiing background)
- Mad River Glen page
- Smuggler's Notch page
- Backcountry Skiing page
- All pages are linked through a navigation bar in the header

## Semantic HTML
- Use of header, nav, main, section, footer
- Headings, paragraphs, lists, images, tables, videos 
- Embedded YouTube video on the Mad River Glen page
- Local video file on the Backcountry page

## CSS Styling
- Consistent pink color scheme
- Styled navigation bar using Flexbox
- Styled table with borders and padding
- Controlled spacing using margin and padding
- Responsive image and video sizing 
- Media query that adjusts layout for smaller screen sizes
